_Configurar las variables segun su cuenta GCP, en el archivo variables.tf_

_Dar permiso al sh_

```
chmod +x aprovisionar.sh
```

_Para crear la instancia_

```
./aprovisionar.sh CREATE
```

_Para eliminar la instancia_
```
./aprovisionar.sh DESTROY
```
